# Proyecto Final - Control 2

Repositorio correspondiente al proyecto final de la asignatura de Control 2.

## Descripción

En este proyecto se desarrollan y analizan diferentes técnicas de control aplicadas a sistemas discretos y continuos utilizando MATLAB/Simulink.

El repositorio incluye:

- Simulación de controladores:
  - P
  - PI
  - PID
  - Controlador en atraso
  - Controlador en adelanto
  - Controlador LQR
  - Controlador RST

- Codigos usados para la operación de la planta:
  - Operación manual
  - Rutina de identificación
  - Implementación de controladores

---

## Objetivos del proyecto

- Analizar el comportamiento dinámico de sistemas de control.
- Diseñar controladores que cumplan especificaciones de desempeño.
- Comparar diferentes estrategias de control.
- Implementar modelos discretos y continuos.
